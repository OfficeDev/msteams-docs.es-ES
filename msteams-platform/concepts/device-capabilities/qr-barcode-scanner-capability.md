## <a name="code-snippet"></a><span data-ttu-id="350e0-203">Fragmento de código</span><span class="sxs-lookup"><span data-stu-id="350e0-203">Code snippet</span></span>

<span data-ttu-id="350e0-204">**Llamada `ScanBarCode()` API** para examinar QR o código de barras con cámara:</span><span class="sxs-lookup"><span data-stu-id="350e0-204">**Calling `ScanBarCode()` API** for scanning QR or barcode using camera:</span></span>

```javascript
const config: microsoftTeams.media.BarCodeConfig = {
  timeOutIntervalInSec: 30};
microsoftTeams.media.scanBarCode((error: microsoftTeams.SdkError, decodedText: string) => {
  if (error) {
    if (error.message) {
      output(" ErrorCode: " + error.errorCode + error.message);
    } else {
      output(" ErrorCode: " + error.errorCode);
    }
  } else if (decodedText) {
    output(decodedText);
  }
}, config);
```
