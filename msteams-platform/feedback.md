# <a name="support-and-feedback"></a>Soporte técnico y comentarios

## <a name="product-support-and-service-issues"></a>Incidencias de soporte técnico y servicio para productos

Use **Centro de administración de Azure** o **Centro de administración de Microsoft 365** para cualquier **problema de carácter crítico para la empresa** que precise soporte técnico basado en SLA.

| **Soporte técnico** | **Contact** |
|:------------|:------------|
| **Incidencias y errores sobre productos de Microsoft Teams**  | Si tiene un contrato de soporte técnico Premier para :::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: Microsoft 365, visite el Centro de administración de Microsoft 365 y use el menú Soporte técnico para [abrir una solicitud de servicio](https://admin.microsoft.com/). |
| **Ayuda y soporte técnico de Azure** | Si tiene una suscripción de pago de Azure, visite el Centro de administración de Azure para [abrir un ticket](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportreq). |
| **Preguntas generales sobre Microsoft Teams** | Si tiene preguntas generales sobre Microsoft Teams, envíe sus preguntas en :::image type="icon" source="assets/icons/microsoft-icon.png" border="false":::[Comunidad de Microsoft Teams](https://answers.microsoft.com/msteams/forum).|

## <a name="developer-community-help"></a>Ayuda de la comunidad de desarrolladores

La comunidad de desarrolladores de Microsoft Teams usa Stack Overflow y Microsoft Q&A para conectarse con otros desarrolladores para idear, obtener clarificaciones y enviar consultas. Además, también puede usar otros contactos o sitios, según el tipo de ayuda de la comunidad necesaria para plantear problemas, limitaciones y preguntas generales.

### <a name="developer-community-forums"></a>Foros de la comunidad de desarrolladores

Publique sus preguntas y ayude a otros miembros de la comunidad compartiendo y respondiendo a preguntas sobre desarrollo de aplicaciones de Microsoft Teams.

| **Canal de la comunidad**|   **Contact**  |
|:---------------------|:---------------|
|**Ayuda de la comunidad** | <ul> <li>:::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false":::[Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams): Use la etiqueta `microsoft-teams` para publicar preguntas. Siga las instrucciones Stack Overflow, como es proporcionar un título descriptivo, una descripción del problema precisa y detalles para reproducir el incidente de que se trate. Las solicitudes de funciones o las preguntas generales están fuera del tema. Si es un usuario nuevo, consulte [Centro de ayuda de Stack Overflow](https://stackoverflow.com/help).</li>  <li>:::image type="icon" source="assets/icons/microsoft-icon.png" border="false":::[Microsoft Q&A](/answers/topics/office-teams-app-dev.html): Nos encanta Stack Overflow y ofrecer un amplio soporte técnico a las preguntas publicadas en Stack Overflow. Use la etiqueta `office-teams-app-dev` para publicar preguntas. Stack Overflow tiene criterios sobre las preguntas adecuadas y [Microsoft Q&A](/answers/topics/office-teams-app-dev.html) tiene una directiva abierta al respecto. </li><ul> |
| **Grupo de chat de la comunidad** | Use el salón de chat de la comunidad informal para :::image type="icon" source="assets/icons/Teams-icon.png" border="false":::[desarrolladores de aplicaciones de Microsoft Teams](https://gitter.im/OfficeDev/MicrosoftTeamsAppDev). Seleccione **INICIAR SESIÓN PARA EMPEZAR A HABLAR**, si inicia sesión por primera vez. Seleccione **UNIRSE A LA SALA**, si ya ha iniciado sesión. |

### <a name="report-issues"></a>Informar sobre problemas

Envíe problemas y formule preguntas generales relacionadas con el SDK y ejemplos mediante los canales de ayuda de la comunidad.

| **Canal de la comunidad** | **Contact** |
|:----------------------|:------------|
| **Interrupciones de la plataforma** | Compruebe si el incidente ya se ha producido en [interrupciones de la plataforma Teams](https://aka.ms/TeamsPlatform/Outages) antes de generar uno. |
| **Ejemplos** | Envíe problemas con ejemplos a los respectivos  [repositorios de ejemplos de Microsoft Teams](https://github.com/OfficeDev/Microsoft-Teams-Samples).|
|  **Incidencias con el kit de herramientas de Teams o el SDK de TeamsFx y CLI** | <ul><li> :::image type="icon" source="assets/icons/GitHub-icon.png" border="false":::  [Incidentes con GitHub](https://github.com/OfficeDev/TeamsFx/issues): cree nuevos incidentes en el repositorio de GitHub de [TeamsFx (Microsoft Teams Framework)](https://github.com/OfficeDev/TeamsFx) para notificar problemas o generar solicitudes de funciones. Se recomienda usar los incidentes de GitHub para las consultas y para recibir ayuda de la comunidad. <li> :::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false":::[Stack Overflow](https://stackoverflow.com/questions/tagged/teams-toolkit): Use la etiqueta `teams-toolkit` para publicar preguntas. Siga las directrices de Stack Overflow, como es proporcionar un título descriptivo, una descripción de problemas concisa y detalles para reproducir el incidente. Las solicitudes de funciones o las preguntas generales están fuera del tema. </li> </ul> |
| **Incidencias o sugerencias del SDK de extensión de bots y mensajes** | Envíe incidentes y solicitudes de funciones al repositorio del SDK para el lenguaje del bot ([C#](https://github.com/Microsoft/botbuilder-dotnet/), [JavaScript](https://github.com/Microsoft/botbuilder-js), o [Python](https://github.com/Microsoft/botbuilder-python)). Publique preguntas sobre procedimientos en [Stack Overflow](https://stackoverflow.com/questions/tagged/botframework%20microsoft-teams) con la etiqueta `botframework` y `microsoft-teams`. |
| **Incidencias o sugerencias sobre Tab SDK** | Envíe incidencias y solicitudes de características al repositorio del SDK para [Biblioteca JavaScript de Microsoft Teams](https://github.com/OfficeDev/microsoft-teams-library-js/issues). Use la etiqueta `microsoft-teams` para publicar preguntas sobre procedimientos sobre el SDK en [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams). |
