# <a name="support-and-feedback"></a>Soporte técnico y comentarios

## <a name="product-support-and-service-issues"></a>Problemas de soporte técnico y servicio del producto

Use **Azure Admin Center** o **Administración de Microsoft 365 Center** para cualquier **problema crítico** para la empresa que necesite soporte técnico basado en acuerdos de nivel de servicio.

| **Soporte técnico** | **Contact** |
|:------------|:------------|
| **Microsoft Teams problemas y errores del producto** | Si tiene un contrato de soporte técnico Premier para :::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: Microsoft 365, visite el Centro de Administración de Microsoft 365 y use el menú Soporte técnico para [abrir una solicitud de servicio](https://admin.microsoft.com/). |
| **Ayuda y soporte técnico de Azure** | Si tiene una suscripción de Azure de pago, visite el Centro de administración de Azure para [abrir una incidencia](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportreq). |
| **Preguntas generales sobre Microsoft Teams** | Si tiene preguntas generales sobre Microsoft Teams, envíe sus preguntas sobre :::image type="icon" source="assets/icons/microsoft-icon.png" border="false"::: [Microsoft Teams Community](https://answers.microsoft.com/msteams/forum).|

## <a name="developer-community-help"></a>Ayuda de la comunidad de desarrolladores

Microsoft Teams comunidad de desarrolladores usan Stack Overflow y Microsoft Q&A para conectarse con otros desarrolladores para idear, obtener aclaraciones y enviar consultas. Además, también puede usar otros contactos o sitios, dependiendo del tipo de ayuda de la comunidad necesaria para plantear problemas, limitaciones y preguntas generales.

### <a name="developer-community-forums"></a>Foros de la comunidad de desarrolladores

Publique sus preguntas y ayude a otros miembros de la comunidad compartiendo y respondiendo a Microsoft Teams preguntas de desarrollo de aplicaciones.

| **canal de Community**|   **Contact**  |
|:---------------------|:---------------|
|**ayuda Community** | <ul> <li>:::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false":::[Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams): use la `microsoft-teams` etiqueta para publicar preguntas. Siga las instrucciones de Stack Overflow, como proporcionar un título descriptivo, una instrucción de problema concisa y detalles para reproducir el problema. Las solicitudes de características o las preguntas generales están fuera del tema. Si es un usuario nuevo, consulte [El Centro de ayuda de Stack Overflow](https://stackoverflow.com/help).</li>  <li>:::image type="icon" source="assets/icons/microsoft-icon.png" border="false":::[Microsoft Q&A](/answers/topics/office-teams-app-dev.html): Nos encanta Stack Overflow y proporcionamos un amplio soporte técnico a las preguntas publicadas en Stack Overflow. Use la `office-teams-app-dev` etiqueta para publicar preguntas. Stack Overflow tiene criterios sobre las preguntas adecuadas y [Microsoft Q&A](/answers/topics/office-teams-app-dev.html) tiene una directiva abierta sobre esto. </li><ul> |
| **Community grupo de chat** | Use el salón de chat de la comunidad informal para :::image type="icon" source="assets/icons/Teams-icon.png" border="false":::[Microsoft Teams desarrolladores de aplicaciones](https://gitter.im/OfficeDev/MicrosoftTeamsAppDev). Seleccione **INICIAR SESIÓN PARA EMPEZAR A HABLAR**, si inicia sesión por primera vez. Seleccione **JOIN ROOM**, si ya ha iniciado sesión. |

### <a name="report-issues"></a>Informar sobre problemas

Envíe problemas y formule preguntas generales relacionadas con el SDK y los ejemplos mediante los canales de ayuda de la comunidad.

| **canal de Community** | **Contact** |
|:----------------------|:------------|
| **Interrupciones de la plataforma** | Compruebe si el problema ya se ha producido en [Teams interrupciones de la plataforma](https://aka.ms/TeamsPlatform/Outages) antes de generar uno. |
| **Muestras** | Envíe problemas con los ejemplos al repositorio de [ejemplos de Microsoft Teams](https://github.com/OfficeDev/Microsoft-Teams-Samples) respectivo.|
|  **problemas del SDK de Teams Toolkit o TeamsFx y la CLI** | <ul><li> :::image type="icon" source="assets/icons/GitHub-icon.png" border="false":::  [problemas de GitHub](https://github.com/OfficeDev/TeamsFx/issues): cree nuevos problemas en el repositorio de [GitHub TeamsFx (Microsoft Teams Framework)](https://github.com/OfficeDev/TeamsFx) para notificar problemas o generar una solicitud de características. Se recomienda usar GitHub problemas para las consultas y recibir ayuda de la comunidad. <li> :::image type="icon" source="assets/icons/stack-overflow-icon.png" border="false":::[Stack Overflow](https://stackoverflow.com/questions/tagged/teams-toolkit): use la `teams-toolkit` etiqueta para publicar preguntas. Siga las instrucciones de Stack Overflow, como proporcionar un título descriptivo, una instrucción de problema concisa y detalles para reproducir el problema. Las solicitudes de características o las preguntas generales están fuera del tema. </li> </ul> |
| **Problemas o sugerencias del SDK de extensión de bots y mensajes** | Envíe problemas y solicitudes de características al repositorio del SDK para el lenguaje del bot ([C#](https://github.com/Microsoft/botbuilder-dotnet/), [JavaScript](https://github.com/Microsoft/botbuilder-js) o [Python](https://github.com/Microsoft/botbuilder-python)). Publique preguntas de procedimientos en [Stack Overflow](https://stackoverflow.com/questions/tagged/botframework%20microsoft-teams) con la `botframework` etiqueta y `microsoft-teams` . |
| **Sugerencias o problemas del SDK de tabulación** | Envíe problemas y solicitudes de características al repositorio del SDK para [Microsoft Teams biblioteca de JavaScript](https://github.com/OfficeDev/microsoft-teams-library-js/issues). Use la `microsoft-teams` etiqueta para publicar preguntas de procedimientos sobre el SDK en [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-teams). |
