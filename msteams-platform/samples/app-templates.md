# <a name="app-templates-for-microsoft-teams"></a>Plantillas de aplicaciones para Microsoft Teams

Las plantillas de aplicación son ejemplos de aplicaciones completas para Microsoft Teams que son de código abierto y están disponibles en GitHub. Cada plantilla de aplicación contiene instrucciones detalladas para implementar e instalar esa aplicación para su organización. También proporciona una aplicación de ejemplo que puede instalar y empezar a usar inmediatamente. El código fuente completo también está disponible, lo que le permite explorarlo en detalle o bifurcar el código y modificarlo para satisfacer sus requisitos específicos.
Todas las plantillas de aplicación se proporcionan bajo los términos [MIT License](https://github.com/OfficeDev/microsoft-teams-apps-eprescription/blob/master/LICENSE).

> [!NOTE]
> Debe conceder licencias y admitir aplicaciones creadas a partir de plantillas de aplicación para los usuarios y las organizaciones.

Las principales ventajas del uso de plantillas de aplicación son:

* **Implemente directamente en la nube:** Todas las plantillas de aplicación incluyen scripts de implementación que permiten hospedar todos los servicios necesarios en Microsoft Azure o Power Platform.
* **Código muestra recomendado:** Las plantillas de aplicación se ajustan a los procedimientos recomendados para la seguridad y la infraestructura. Todos los cambios enviados por la comunidad a las plantillas de aplicación se revisan para garantizar la conformidad.
* **Personalizable y extensible:** Aunque todas las plantillas de aplicación se implementan con una configuración mínima, se proporcionan scripts de implementación y código base completos, para que pueda personalizarlos o ampliarlos fácilmente para adaptarlos a sus necesidades únicas.
* **Documentación detallada:** Todas las plantillas de aplicación van acompañadas de documentación completa sobre la arquitectura de la solución, la implementación y los pasos de configuración.  

## <a name="champion-management-platform"></a>Plataforma de administración de expertos

La plantilla de aplicación Plataforma de administración de expertos (CMP) le ayuda a administrar, escalar e inspirar a sus expertos de trabajo en equipo para lograr más. Esta plantilla de aplicación se basa en SharePoint Framework y se carga en una pestaña dentro de un equipo. Los grupos pueden aprovechar esta herramienta para ayudar a administrar la pertenencia al programa, proporcionar una tabla de clasificación y tipos de eventos para el registro, y herramientas para superponer distintivos digitales a los participantes del programa.

[Obténgalo en GitHub](https://github.com/OfficeDev/microsoft-teams-apps-champion-management)

## <a name="company-communicator"></a>Comunicador de la empresa

La aplicación Comunicador de empresa permite a los equipos corporativos crear y enviar mensajes destinados a varios equipos o a un gran número de empleados a través del chat, lo que permite a la organización llegar a los empleados justo donde colaboran. Use esta plantilla para varios escenarios, como anuncios de nuevas iniciativas, incorporación de empleados, aprendizaje moderno y difusión de desarrollo o de toda la organización.

La aplicación proporciona una interfaz sencilla para que los usuarios designados creen, obtengan una vista previa, colaboren y envíen mensajes.

Proporciona una base para crear funcionalidades de comunicación dirigida personalizadas, como la telemetría personalizada, sobre cuántos usuarios han confirmado o interactuado con un mensaje.

[Obténgalo en GitHub](https://github.com/OfficeDev/microsoft-teams-company-communicator-app)

## <a name="faq-plus"></a>Preguntas más frecuentes Plus

Preguntas y respuestas conversacionales Los bots son una manera sencilla de proporcionar respuestas a las preguntas más frecuentes de los usuarios. Sin embargo, la mayoría de los bots no pueden interactuar con los usuarios de forma significativa porque no hay ningún humano en el bucle cuando se produce un error en el bot. El bot de preguntas frecuentes es un bot Q&A descriptivo que pone a un humano en el bucle cuando no puede ayudar. Se puede formular una pregunta al bot y el bot responde con una respuesta si está incluido en el knowledge base. Si no es así, el bot permite al usuario enviar una consulta, que luego se publica en un equipo preconfigurado de expertos que ayudan a proporcionar soporte técnico actuando sobre las notificaciones desde el propio equipo.

> [!NOTE]
> La versión más reciente de **FAQ Plus** admite preguntas y respuestas mejoradas Una solución que permite a un equipo de expertos completar lo siguiente:
>
> &#x2714; Agregar nuevas preguntas y respuestas Como directamente a la Knowledge Base mediante extensiones de mensaje.
>
> &#x2714; Editar y eliminar preguntas y respuestas Pares agregados por un bot.
>
> &#x2714; Realice un seguimiento del historial de revisiones de Preguntas y respuestas.
>
> &#x2714; Configure una respuesta con detalles adicionales para que se muestre como una [Tarjeta adaptativa](../task-modules-and-cards/cards/cards-reference.md#adaptive-card).
>
[Obténgalo en GitHub](https://github.com/OfficeDev/microsoft-teams-apps-faqplusv2)

## <a name="icebreaker"></a>Rompehielo

Icebreaker es un [Bot de Microsoft Teams](../bots/what-are-bots.md) que ayuda a su equipo a acercarse emparejando dos miembros aleatorios del equipo cada semana para reunirse. El bot facilita la programación al sugerir automáticamente horas libres que funcionen para ambos miembros. Refuerce las conexiones personales y cree una comunidad estrechamente estrecha con esta aplicación.

Además de fomentar conexiones personales en todo el equipo, la aplicación Icebreaker puede ayudar a fomentar comunidades basadas en intereses dentro de la organización. Por ejemplo, puede usar esta aplicación para un grupo de interés de DevOps con el fin de ayudar a que las ideas y los procedimientos recomendados se distribuyan de forma orgánica en toda la organización.

[Obténgalo en GitHub](https://github.com/OfficeDev/microsoft-teams-icebreaker-app)

## <a name="microsoft-365-learning-pathways-get-started"></a>Caminos de aprendizaje de Microsoft 365 (introducción)

La plantilla de aplicación Introducción le permite aportar el poder de los caminos de aprendizaje de Microsoft 365 dentro de Teams. Esta plantilla de aplicación le permite conceder acceso fácil a páginas de aprendizaje específicas u otros recursos de intranet y cargar el contenido directamente en Teams. También puede cambiar el nombre o el logotipo de la aplicación para que coincida con la personalización de marca de su empresa.

[Obténgalo en GitHub](https://github.com/msft-teams/tools/tree/master/M365%20Learning%20Pathways)

## <a name="microsoft-teams-emergency-operations-center-teoc"></a>Centro de operaciones de emergencia (TEOC) de Microsoft Teams

La plantilla de solución Del Centro de operaciones de emergencia (TEOC) de Microsoft Teams aprovecha la eficacia de la plataforma de Microsoft 365 para centralizar la respuesta a incidentes, el uso compartido de información y las comunicaciones de campo mediante servicios eficaces como Listas Microsoft, SharePoint y mucho más. Una solución de código abierto compatible con Microsoft, proporciona funcionalidad básica y se puede ampliar para satisfacer los requisitos específicos de la agencia.

[Obténgalo en GitHub](https://github.com/OfficeDev/microsoft-teams-emergency-operations-center/)
